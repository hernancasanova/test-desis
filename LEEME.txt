1) Versión de PHP: 8.1.21. Versión de Mysql: 5.7
2) Software requerido:
    -docker y docker-compose.
3) Pasos para ejecutar la aplicación: 
    -Dentro del directorio del proyecto levantar contenedores de la aplicación con docker-compose con el comando: docker-compose up.
    -Esperar algunos minutos para asegurarse que el script de creación de la base de datos se ejecute correctamente.
    -Abrir el archivo form.html y probar la aplicación.
